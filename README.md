# HERFISH LEGACY - Getting Started

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Configure environment
cd backend
cp .env.example .env
cd ../frontend
cp .env.example .env

# 3. Start development
npm run dev
```

**Frontend:** http://localhost:5173  
**Backend:** http://localhost:3000

## Project Structure

```text
HERFISH-LEGACY/
|-- frontend/          # React + Vite
|-- backend/           # Express API
`-- docs/              # Documentation
```
